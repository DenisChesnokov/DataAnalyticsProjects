|Название проекта| Описание| Используемые технологии и библиотеки |

|[Прогнозирование вероятности оттока пользователей для фитнес-центров](https://github.com/DenisChesnokov/DataAnalyticsProjects/blob/main/fitness_club_customer_churn_prediction/fitness_club_customer_churn_prediction.ipynb)|Проект на основе алгоритмов машинного обучения. В проекте спрогнозирована вероятность оттока для каждого клиента на один месяц вперёд. На основе поведени клиентов сформированы портреты пользователей, выделены группы, охарактеризованы их основные свойства и выявлены признаки влияющие на вероятность оттока в следующем месяце. Даны рекомендаци по уменьшению оттока.| Python, Pandas, Scikit-learn, Matplotlib, Seaborn, машинное обучение, классификация, кластеризация|
